常见问题

Telegram 怎么隐藏手机号不被好友看到？ ▼
进入 Telegram 设置 → 隐私和安全 → 手机号码，将「谁可以看到我的手机号码」设置为「没有人」，并将「谁可以通过手机号码找到我」设为「我的联系人」。详细步骤请查看 Telegram 隐私安全设置完全指南。
Telegram 两步验证怎么开启？有什么用？ ▼
在设置 → 隐私和安全 → 两步验证中设置密码，可以防止他人仅凭短信验证码登录你的账号。即使 SIM 卡被克隆，没有两步验证密码也无法登录。详见 Telegram 两步验证完整教程。
Secret Chat 和普通聊天有什么区别？ ▼
Secret Chat 采用端到端加密（E2E），消息仅存在双方设备上、不经过 Telegram 服务器中转；支持阅后即焚、禁止转发和截图提醒。普通群聊和频道使用的是服务器端加密。详情见 Secret Chat 端到端加密原理。
Telegram 账号被盗了怎么找回？ ▼
首先在活跃会话中强制下线陌生设备，然后重置两步验证密码，最后联系 Telegram 官方支持。关键信号：收到未授权的登录验证码、好友收到你的异常消息、设备列表出现未知设备。完整步骤见 Telegram 账号被盗找回指南。
怎么查看有哪些设备登录了我的 Telegram？ ▼
设置 → 隐私和安全 → 活跃会话，可以看到所有登录设备及其 IP、地区和最后活跃时间。点击任一设备可远程注销。建议每两周检查一次，不认识的设备立即踢掉。详见 活跃会话管理教程。
Telegram 和 WhatsApp、Signal 哪个最安全？ ▼
Signal 默认端到端加密最强；WhatsApp 也是 E2E 但属于 Meta（隐私争议较大）；Telegram 只有 Secret Chat 是 E2E，普通聊天为服务器端加密，但功能最丰富、跨平台体验最好。三者各有优劣，完整对比见 Telegram vs WhatsApp vs Signal 深度对比。
